# 🍞 Bread & Butter Bakery Website

A modern, responsive bakery website built using **HTML, CSS, and JavaScript**.
The website showcases bakery products, menu, gallery, and allows users to place orders through a simple interactive cart system.

---

## 📌 Features

* Responsive design for **desktop, tablet, and mobile**
* Beautiful **hero section with image slider**
* Interactive **menu page**
* **Add to cart** system with quantity controls
* Order summary with **total price calculation**
* **Gallery with fullscreen image preview**
* FAQ section with **accordion effect**
* Animated sections using **CSS animations**
* Modern **responsive navigation menu**
* Attractive **footer with contact form and social links**

---

## 🛠 Technologies Used

* **HTML5**
* **CSS3**
* **JavaScript (Vanilla JS)**
* **Flexbox & Grid Layout**
* **Responsive Media Queries**
